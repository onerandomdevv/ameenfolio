## ‚ú® Key Features

### üé® "Electric Lime" Design System

A custom-built design language featuring a stark black/zinc palette accented by a vibrant `#CCFF00` (Electric Lime).

- **Glassmorphism**: Extensive use of backdrop blurs and semi-transparent layers for depth.
- **Glitch Effects**: Custom `GlitchText` components for dynamic headers.
- **Ken Burns Hero**: A cinematic hero section with subtle zoom and pan effects.

### üì± "Pure Perfection" Responsiveness

Optimized for every device, from the smallest iPhone SE to the largest 4K monitors.

- **Adaptive Navigation**: Transforms from a desktop "pill" navbar to a mobile friendly hamburger menu with smooth overlays.
- **Fluid Typography**: Text scales dynamically using variable fonts and responsive utilities.
- **Stacking Grids**: Complex layouts (like the Tech Stack and Expertise sections) elegantly stack on mobile without losing context.
- **Touch-Optimized**: Horizontal sliders and interactive elements are tuned for swipe gestures.

### ‚ö° Performance First

- **0 Layout Shift**: Images and fonts are preloaded and sized to prevent CLS.
- **Lazy Loading**: Components and heavy assets load only when needed.
- **Semantic HTML**: Built with accessibility and SEO best practices from the ground up.

## üõ†Ô∏è Getting Started

If you'd like to run this project locally or use it as a reference:

1.  **Clone the repository**:

    ```bash
    git clone https://github.com/onerandomdevv/ameen-s-portfolio.git
    cd ameen-s-portfolio
    ```

2.  **Install dependencies**:

    ```bash
    npm install
    # or
    yarn install
    ```

3.  **Run the development server**:

    ```bash
    npm run dev
    ```

4.  **Open your browser**:
    Navigate to [http://localhost:3000](http://localhost:3000) to view the application.
